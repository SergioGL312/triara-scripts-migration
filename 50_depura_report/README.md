# Script 50_depura_report

## Descripción

Este script automatiza la eliminación de archivos de backup antiguos en directorios específicos, tanto para bases de datos Oracle como MySQL. Los archivos son identificados por un prefijo en su nombre y se eliminan si son más antiguos de 6 días, conservando solo una cantidad máxima definida. El código fue traducido a Python desde su versión original para mayor eficiencia y claridad.

# No external packages required. Only standard library is used.
